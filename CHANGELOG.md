# Changelog

All notable changes to MerchantMind AI will be documented in this file.

This project follows Semantic Versioning (SemVer).

---

## [v0.3.0-alpha.1]

### Added

- Review Analysis Agent
- Sales Analysis Agent
- Inventory Analysis Agent
- Google Gemini integration
- Jinja2 prompt templating
- Sunrise Café demo datasets
- Sales, Review and Inventory data loaders
- LangGraph multi-agent workflow
- Centralized LLM service
- Structured BusinessState
- Logging and global exception handling

### Changed

- Migrated prompts from Markdown to Jinja2 templates
- Standardized agent architecture
- Standardized JSON-based state objects
