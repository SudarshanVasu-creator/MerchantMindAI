### The MVP Scope



We'll intentionally avoid building anything that isn't needed for the demo.



#### Included

React dashboard

FastAPI backend

LangGraph

Multi-agent orchestration

OpenRouter LLM

Charts

Report generation

Sample datasets





#### Excluded

Authentication

Payments

Real Google integration

Real WhatsApp integration

Multi-tenancy

PostgreSQL (for now)

Redis

Docker deployment



We'll simulate integrations with well-structured sample data so we can focus on the AI workflow.
